About Us

e5movies is a website which provides information about released or upcoming movies such as their storyline, cast and crew, director, etc. along with the official trailer from IMDB website.


This Web Application was built in Angular CLI model using :

  • Angular 6 - Angular CLI (Front-End)
  • Express (Back-End Server)
  • Mongodb (Database)
  • Nodejs (Layout for Express)
  • Bootstrap (Designing)
  • Angular Material (Designing)
  • Visual Studio Code (Code Editor)
